# A Large-Scale Dataset for Real Estate Hotspot Prediction

This repository contains the implementation of our paper, "Modeling Real Estate Dynamics Using Temporal Encoding", ACM SIGSPATIAL 2021.

## Setup
### Required Packages
- Numpy
- Pandas
- matplotlib
- scikit-learn
